# FITTORY 메인페이지

React, Vite 및 별도 프레임워크 없이 HTML, CSS, JavaScript로 구성된 정적 쇼핑몰 메인페이지입니다.

## 폴더 구조

- `index.html` — 페이지 콘텐츠와 섹션 구조
- `css/style.css` — 전체 디자인 및 반응형 스타일
- `js/script.js` — 모바일 메뉴, 장바구니, 찜하기, 뉴스레터 기능
- `img/` — 페이지에 사용되는 모든 이미지

## 수정 방법

`index.html`을 브라우저에서 열면 바로 확인할 수 있습니다. 문구와 상품은 `index.html`, 색상과 배치는 `css/style.css`, 동작은 `js/script.js`에서 수정하세요.
