# VIONA 메인페이지

VIONA는 건강기능식품을 판매하는 라이프스타일 웰니스 브랜드입니다. 건강 콘텐츠와 맞춤형 상품 추천을 통해 누구나 자신의 건강 루틴을 쉽고 꾸준히 실천할 수 있도록 돕습니다.

**Brand Slogan:** Healthy Routine, Better Everyday.

이 프로젝트는 React, Vite 및 별도 프레임워크 없이 HTML, CSS, JavaScript로 구성된 정적 브랜드 쇼핑몰 메인페이지입니다.

## 폴더 구조

- `index.html` — 페이지 콘텐츠와 섹션 구조
- `css/style.css` — 전체 디자인 및 반응형 스타일
- `css/navigation.css` — GNB, LNB, 드롭다운, 모바일 메뉴 스타일
- `js/script.js` — 모바일 메뉴, 장바구니, 찜하기, 뉴스레터 기능
- `js/navigation.js` — PC/모바일 내비게이션 및 접근성 인터랙션
- `img/` — 페이지에 사용되는 모든 이미지

## 수정 방법

`index.html`을 브라우저에서 열면 바로 확인할 수 있습니다. 내비게이션의 색상과 간격은 `css/navigation.css`, 메뉴 동작은 `js/navigation.js`에서 수정하세요. 나머지 페이지 디자인과 기능은 `css/style.css`, `js/script.js`에서 관리합니다.
